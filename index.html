Wrestling Hand Signal Classifier

A Teachable Machine project that helps wrestling referees identify and practice official hand signals.

About the Project

This tool uses a machine learning model trained through Teachable Machine to recognize common wrestling referee hand signals. When you perform a pose, the system detects the gesture and displays which hand signal you are doing, along with a confidence percentage.

This can help new referees practice, improve consistency, and learn proper form.

How It Works

The model takes your webcam feed, uses PoseNet to detect body keypoints, and then classifies those keypoints into hand-signal classes such as “Start Match,” “Stop Match,” and “Near Fall.”
